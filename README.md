# ⚔️ 코테를 부수겠다는 마음으로 나타난 전사들 ⚔️

## ⛳️ 목표
**코딩 괴물이 되는 그날까지!!**  

_코테 부서벼려_ 👊


## 🧵 스터디 규칙
- 매주 목요일 저녁 8시
- 이슈에 등록된 문제를 풀이한 뒤 인증샷 코멘트 남기기
- 각 문제에 대해 설명하고 알고리즘 설명하기(30분)
- 랜덤 한 문제 풀기 (30분, 라이브 코딩)
